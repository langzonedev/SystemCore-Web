# SystemCore Web

Public browser edition of [SystemCore](https://github.com/langzonedev/SystemCore).

Open the live application at:

<https://langzonedev.github.io/SystemCore-Web/>

## Data storage and privacy

SystemCore is a static browser application with no server-side database. Project
data is stored in the current browser's `localStorage` and is not uploaded to
GitHub or shared with other users.

- Each browser and device has its own separate project data.
- Clearing browser/site data removes the locally stored project.
- Data does not automatically synchronize between devices.
- Use **Export JSON** regularly for backup and transfer.

Do not rely on browser storage as the only copy of important project data.
